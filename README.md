# 🦊 茉莉的美味 Lab

家庭共享的"明日两餐"规划工具，主厨：茉莉。

- **四大料理**：A 忙忙料理（偷懒）· B 瘦瘦料理（控卡）· C 大理人的秘密欢愉（丰盛/云南）· D 躺平（不做饭）
- **一天两餐**：午餐 + 晚餐，各选类别 → 抽卡/点选菜品，全家实时同步
- **冰箱**：记录买回的菜（带日期、快过期提醒）；某顿「做完了」自动划掉用到的食材
- **灵感**：按冰箱现有食材推荐能做的菜
- **买菜清单**：按计划的菜 + 人数（默认4人）自动算出需买食材
- **卡路里**：按食材 + 做法自动估算（约 kcal/人份）

技术：纯静态 `index.html` + Supabase（实时同步）+ GitHub Pages。MIT。
